Mission 1

Mission Objective : Through mission step 1~3, you will connect to a local Ethereum network and perform mining process. You will also create your own Eth account and remove them.
 
Mine Ethereum within a local network using a pre-built account. Once you have successfully mined Ethereum, deploy a simple Smart Contract using a Remix IDE. To complete the mission, create a basic DApp which you will use to update Smart Contract content.

Mission Objective : we will experience various blockchain applications running on the Private Network created in Mission2. For example Ethereum monitoring, Crowdfunding DApp based on ERC20 token, data analysis using chain graph and wallet etc.
