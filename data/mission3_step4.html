Note

Blockchain transaction Monitoring and Analytics based on Graph DB
- if you represent blockchain transactions in graph,
  ･ The first vertex is the sender account, the second vertex is the receiver account,
  ･ The middle edge is the transaction and has the transferred value as the property of edge.
- AWS Services : Neptune Graph DB

- If you apply this to the ethereum Main Net,
  ･ cryptocurrency exchange accounts : have the most in-out transactions
  ･ fraud accounts : show abnormal patterns
    (ex. Ransomware pattern: the same value is transfered from multiple accounts to the one account)
